Simple by design

Three lines of code to generate your first video. No complex setup required.

generate.js 
import { VideoLLMClient } from 'videollm-sdk';

const client = new VideoLLMClient({
  baseURL: 'https://prototype-qong.onrender.com',
  apiKey:  'your-api-key'
});

// Generate video with smart routing
const result = await client.generate({
  prompt: 'A drone shot over neon Tokyo at night',
  strategy: 'best-quality',  // or 'best-value'
  resolution: '4k'
});

console.log(result.videoUrl);
// → https://...output.mp4
